# 🌴 Indo Travel

> Лендинг туристической компании по путешествиям в Индонезию

🔗 **[Живой демо](https://usmankamayl.github.io/indo-travel/)**

---

## 📋 О проекте

Indo Travel — одностраничный сайт (landing page) туристической компании, специализирующейся на турах по островам Индонезии. Сайт охватывает полный цикл взаимодействия с клиентом: от знакомства с направлением до оформления заявки на бронирование.

---

## ✨ Функциональность

### 🎠 Слайдеры
- Слайдер фотографий в разделе «Об Индонезии»
- Фотоальбом с галереей снимков из поездок
- Плавная навигация между слайдами

### 📅 Динамическое ценообразование
- Цена тура рассчитывается **в реальном времени** в зависимости от:
  - Выбранной даты путешествия
  - Количества туристов в группе
  - Дополнительных опций
- Мгновенное обновление стоимости при изменении параметров

### 📬 Форма бронирования
- Поля: дата путешествия, количество человек, ФИО, телефон
- Валидация данных перед отправкой
- Отправка заявки с обратной связью для пользователя

### 🔔 Уведомления
- **Успешная отправка** — красивый попап с подтверждением и сообщением о сроках ответа менеджера
- **Ошибка отправки** — информативное сообщение с предложением повторить попытку

### 🪗 Аккордеоны
- Раздел «Путешествие» с маршрутом по дням — реализован в виде аккордеона
- Плавное раскрытие/скрытие контента

### ⏱️ Таймер обратного отсчёта
- Отображает время до окончания акции (скидка 20% на туры от 6 человек)
- Обновляется в реальном времени: дни, часы, минуты

### 📱 Адаптивный дизайн
- Корректное отображение на мобильных устройствах, планшетах и десктопе
- Мобильное меню

## 🗺️ Структура страницы

| Раздел | Описание |
|---|---|
| **Hero** | Главный экран с заголовком, таймером акции и CTA-кнопкой |
| **Индонезия** | Описание страны со слайдером фотографий |
| **Путешествие** | Маршрут тура по дням в формате аккордеона |
| **Тур 14 дней** | Калькулятор стоимости с выбором даты и кол-ва человек |
| **Почему мы?** | 6 конкурентных преимуществ компании |
| **Фотоальбом** | Галерея фотографий из поездок |
| **Бронирование** | Форма подачи заявки |
| **Контакты** | Телефон, email, соцсети и форма обратной связи |

## 🏝️ Направления тура

- **День 1–2** — Джакарта (о. Ява)
- **День 3–5** — о. Бали
- **День 6–8** — о. Сулавеси
- **День 9–11** — о. Калимантан (Борнео)
- **День 12–14** — о. Суматра

## 🛠️ Технологии

- **HTML5** — семантическая разметка

## 🚀 Запуск проекта

```bash

git clone https://github.com/usmankamayl/indo-travel.git

cd indo-travel
```
